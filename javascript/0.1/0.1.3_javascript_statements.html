JavaScript Statements:

JavaScript statements are the building blocks of a JavaScript program. Each statement performs a specific action. Examples of JavaScript statements include variable declarations, loops, function calls, and more.

JavaScript Fundamentals:

  • Variables and Declarations
  • Data Types
  • Operators
  • Conditional statements (if, else, switch)
  • Loops (for, while, do...while)
  • Functions
  • Error handling (try...catch)

Variables and Declarations:

In JavaScript, you can declare variables using var, let, or const. Variables are used to store data. For example:

let age = 30; const name = "John";

Data Types:

JavaScript supports various data types, including numbers, strings, booleans, objects, arrays, and more. For example:

let number = 42; let greeting = "Hello, World!"; let isReady = true;

Conditional Statements:

Conditional statements allow you to make decisions in your code. Common conditional statements are if, else if, else, and switch. They help you execute different code blocks based on specified conditions. For example:

if (age >= 18) { console.log("You are an adult."); } else { console.log("You are a minor."); }

Loops:

Loops are used to repeatedly execute a block of code. Common loops in JavaScript are for, while, and do...while. They help automate repetitive tasks.

for (let i = 0; i < 5; i++) { console.log("Iteration " + i); }

Functions:

Functions are reusable blocks of code that perform a specific task. You can define your own functions and call them when needed.

function greet(name) { return "Hello, " + name + "!"; } let message = greet("John"); console.log(message);

Error Handling (try...catch):

JavaScript allows you to handle errors gracefully using try...catch blocks. This helps prevent unexpected issues from crashing your program.

try { // Code that might throw an error let result = 10 / 0; } catch (error) { console.error("An error occurred: " + error.message); }
